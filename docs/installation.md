# Installation
## From source

The source files for Ontario Data Centres can be downloaded from the [Github repo](https://github.com/mothra729/ontario_data_centres).

You can either clone the public repository:

```sh
git clone https://github.com/mothra729/ontario_data_centres
```

Or download the [tarball](https://github.com/mothra729/ontario_data_centres/tarball/main):

```sh
curl -OJL https://github.com/mothra729/ontario_data_centres/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd ontario_data_centres
uv sync
```
